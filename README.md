# Projeto Filmes

## Descrição
Este é um projeto frontend que lista filmes utilizando uma API pública para busca e exibição de informações como título, sinopse, avaliação e imagem. A interface foi desenvolvida com Vue.js e Tailwind CSS para garantir uma experiência rápida e responsiva.

---

## Tecnologias e Versões

- **Node.js**: v18.16.0  
- **Yarn**: 1.22.19  
- **Framework**: Vue.js 3  
- **Estilização**: Tailwind CSS  

---

## API utilizada

Este projeto consome a API [The Movie Database (TMDb)](https://www.themoviedb.org/documentation/api), que fornece dados atualizados sobre filmes, séries e atores.

---

## Pacotes principais utilizados

- `axios` — para fazer requisições HTTP à API de filmes  
- `vue-router` — para navegação entre páginas dentro do app  
- `pinia` (ou `vuex`) — para gerenciamento de estado (se usado)  
- `tailwindcss` — framework CSS para estilização responsiva e moderna  
- `vue` — framework JavaScript progressivo para a interface  
- `@vueuse/core` (opcional) — utilitários Vue para facilitar algumas operações
